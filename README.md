# 1c-AI-assistant

Этот проект — пример и основа для интеграции систем на платформе "1С:Предприятие" с AI-моделями Google Gemini.

Он демонстрирует, как создать AI-агента на языке Go, который может использовать инструменты для получения данных из системы, имитирующей 1С.

## Основные возможности

*   **AI-агент на Go**: Использует [Google ADK (Agent Development Kit)](https://github.com/google/genkit/tree/main/go/adk) для создания агента.
*   **Интеграция с Gemini**: Настроен для работы с моделью `gemini-1.5-flash-001`.
*   **Инструменты для 1С**: Реализованы два примера инструментов:
    *   `get_stock_balance` — для получения остатка товара на складе.
    *   `get_counterparty_debt` — для получения задолженности контрагента.
*   **Имитация базы данных 1С**: Включает в себя простую имитацию базы данных для демонстрации работы без необходимости подключения к реальной 1С.

## Запуск проекта

1.  **Установите переменные окружения**:
    Для работы с Gemini вам необходим API-ключ.

    ```bash
    export GOOGLE_API_KEY="ВАШ_API_КЛЮЧ"
    ```

2.  **Запустите приложение**:
    Выполните следующую команду в терминале:

    ```bash
    go run .
    ```

    После запуска будет доступен веб-интерфейс для взаимодействия с агентом.

## Примеры работы

Агенту можно задавать вопросы на естественном языке. Он сам выберет нужный инструмент и вернет результат.

**Пример 1: Узнать остаток товара**

> **Пользователь:** Сколько стульев офисных стандарт на складе?
>
> **Агент:** На складе 312 шт. товара "стул офисный стандарт".

**Пример 2: Узнать задолженность**

> **Пользователь:** Какая задолженность у ООО Ромашка?
>
> **Агент:** Задолженность клиента "ооо ромашка" составляет 125430.50.

**Пример 3: Нечеткий поиск**

> **Пользователь:** остаток по столам
>
> **Агент:** (после поиска) Уточните, какой именно стол вас интересует? Например, "стол офисный модель а".
